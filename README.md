# Welcome to my final project for HCDE 310!

  This program allows you to input a location and gives you the current weather conditions, temperature, and an artwork from the MET that 
  matches the current conditions. If you include more than just the city name, such as the state, it will display a fuller address. For example,
  typing in Hanoi,Vietnam, will display Hà Nội, Việt Nam. Use it for your daily art inspirational needs, or if you want to see artwork you've never seen from the MET's
  database. It'll link you to the MET's website, which has more information about the art piece!

## API Key
It uses the Visual Crossing API, which requires a key, for the weather information. To start this, create a free account on their
website (https://www.visualcrossing.com/sign-up/) and create a hidden file called keys.py with a variable called vkey with the key generated 
with your account. It should work after that! Alternatively, there is a pythonanywhere link currently that has this program running on it if you
wish not to go through this process: https://greenbamboo.pythonanywhere.com/.
